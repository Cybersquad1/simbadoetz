Menu Koreksi


Sub menu ini digunakan untuk memperbaiki atau menghapus data yang sudah direkam, apabila terjadi kesalahan input atau kesalahan lainnya. Akses sub menu ini hanya dibatasi oleh pejabat tertentu saja


Mempunyai 2 alur pengoperasian, yaitu:


1.


Tahapan untuk melakukan koreksi data adalah sebagai berikut:


1. Pilih dan klik sub menu koreksi data aset.


2. SIMBADA akan menampilkan halaman koreksi data aset yang berisi form pencarian.

3. Isikan seluruh informasi aset yang tersedia dengan lengkap dan benar.


4. Jika ingin menampilkan seluruh daftar aset tanpa filter (kosongkan form isian). kemudian pilih tombol lanjut, maka akan tampil pesan seperti ini.


5. SIMBADA akan menampilkan halaman koreksi data berupa informasi aset. Pilih dan klik tombol EDIT DATA pada aset yang ingin diubah, untuk mengubah informasi aset.


6. Perbaiki data aset yang diperlukan pada bagian informasi aset, mulai dari informasi umum, SKPD, jenis Barang, Lokasi, Cara Perolehan aset, Penghapusan Aset,hingga Pemeriksaan.


7. Klik Tombol simpan untuk menyimpan data.

8. Klik tombol hapus untuk menghapus data.



2.


Tahapan untuk melakukan Cetak daftar aset adalah sebagai berikut:


1. Pilih dan klik sub menu koreksi data aset.


2. SIMBADA akan menampilkan halaman koreksi data aset yang berisi form pencarian.

3. Isikan seluruh informasi aset yang tersedia dengan lengkap dan benar.


4. Jika ingin menampilkan seluruh daftar aset tanpa filter (kosongkan form isian). kemudian pilih tombol lanjut, maka akan tampil pesan seperti ini.


5. SIMBADA akan menampilkan halaman koreksi data berupa informasi aset. Pilih dan klik tombol EDIT DATA pada aset yang ingin diubah, untuk mengubah informasi aset.
